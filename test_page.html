🆕 New Feature: Editable AI Prompt Dialog!

Select any question text below and you'll see a 🔍 magnifying glass icon appear above it.

Click the icon to open a draggable, resizable dialog with an editable AI prompt!

Default prompt: "Send me the answer and option number without explanation" - but you can customize it!

📝 How to Test the New Dialog:

  • Method 1 (NEW): Select question text → Click 🔍 icon → Edit prompt & send
  • Method 2: Select question text → Right-click → "Solve with AI"
  • Method 3: Select text → Press Ctrl+Shift+Q

💡 Dialog Features:

  • 📝 Editable Prompt: Customize what you ask the AI
  • 🖱️ Draggable: Click and drag the header to move the dialog
  • 📏 Resizable: Drag the bottom-right corner to resize
  • ➖ Minimizable: Click the minimize button to collapse
  • 🔄 Reset: Restore the default prompt anytime

🔧 Troubleshooting:

  • If the 🔍 icon doesn't appear, make sure you've selected enough text (5+ characters)
  • If the dialog doesn't open, try clicking the icon again or use right-click method
  • If no AI response comes, check that your Gemini API key is set up correctly in the extension popup
  • If you see "Extension was reloaded" message, simply refresh the page to continue using the extension
  • The dialog is draggable by clicking and dragging the blue header bar
  • You can resize the dialog by dragging the bottom-right corner
  • Use the minimize button (−) to collapse the dialog temporarily
